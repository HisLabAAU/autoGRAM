# autoGRAM

This is the readme for autoGRAM - a pipeline for automated graphing of entities. 

The graphing pipelin consists of steps. 

1. NER.
2. Processing of NER.
3. LLM-queries using DSPY.
4. Graph.

Each step is a separate file.

Howeer. We also include a different version (FULL_STACK.qmd) which performs the part-of-speech-tagging using an LLM. This is a good place to start.
